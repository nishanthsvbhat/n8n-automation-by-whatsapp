üöÄ Quick Start Guide

1. Install n8n

npm install n8n -g

2. Configure Environment Variables

Copy .env.example to .env and fill in your values:

cp .env.example .env
Required: You need WhatsApp Business API credentials from Facebook Developer Console

3. Start n8n

n8n start

4. Import n8n Workflows

Import the provided workflows in the n8n-workflows/ directory

5. Test WhatsApp Integration

  • Send "MENU" to your WhatsApp Business number
  • Try placing an order: "1x2, 3x1"
  • Confirm with "CONFIRM"
  • Track with "TRACK"
